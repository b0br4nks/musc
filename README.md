## Usage
```console
# Simulate the program
$ ./musc.py -s <path/to/file.mxc>

# Compile the program (Assembly code)
$ ./musc.py -c <path/to/file.mxc>

# Run the compiled program
$ ./output
```
Or simply:
```console
$ make
```
